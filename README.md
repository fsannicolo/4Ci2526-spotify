## Installazione Angular
```bash
npm install -g @angular/cli
```

> [!important]
> Angular va installato in global mode `-g`

## Creare una nuova applicazione
```bash
ng new nome-app
```

> [!TIP]
> Con la CLI Angular si consiglia l'utilizzo della notazione kebab-case
